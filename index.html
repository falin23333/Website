Students Performance

Este proyecto se centra en la predicción de calificaciones de exámenes utilizando el Huber Regressor, un modelo de regresión robusto capaz de manejar outliers. Agricultural Yield Prediction

Dataset: 28.242 registros con datos de producción agrícola y factores ambientales en múltiples países. Características: precipitación, temperatura, uso de pesticidas, tipo de cultivo y área geográfica. Objetivo: predecir el rendimiento de los cultivos (hg/ha) utilizando técnicas de Machine Learning. Insights: análisis del impacto de las condiciones climáticas y ambientales en la productividad agrícola, permitiendo identificar patrones clave que influyen en el rendimiento de los cultivos.

Arquitecturas Big Data

Flask + Celery + Redis + Supabase Este proyecto pone en práctica los fundamentos básicos de una arquitectura Big Data y sistemas escalables, enfocándose en la gestión eficiente de grandes volúmenes de información sin saturar la base de datos. Se aplica una arquitectura asíncrona y desacoplada para reducir latencia y evitar cuellos de botella. La solución se basa en Flask como servidor web, Redis como sistema de caché y cola FIFO, Celery para el procesamiento asíncrono de tareas y Supabase (PostgreSQL) como base de datos para la persistencia final de los datos.

Predict My Car 1.0

Predict My Car clasifica la marca de un coche a partir de una imagen utilizando visión por computador. El proyecto emplea una red neuronal convolucional (CNN) entrenada con más de 7.700 imágenes de seis marcas distintas, aplicando preprocesado y aumento de datos. Incluye evaluación mediante matrices de confusión, métricas de clasificación y curvas de rendimiento, alcanzando una precisión aproximada del 86 % en el conjunto de test.

Proyecto práctico utilizando la API de Geotab para la visualización de datos en tiempo real. En este proyecto analizo kilómetros recorridos fuera de horario, dentro de la jornada laboral, errores de los vehículos y su ubicación. Loan Predictor

Predicción de Riesgo Hipotecario. Proyecto de Machine Learning orientado a predecir si un cliente pagará su deuda bancaria. Tras un análisis EDA y el tratamiento del fuerte desbalanceo de clases, se entrenaron y evaluaron múltiples modelos. El modelo final seleccionado fue AdaBoost, integrado en un pipeline de preprocesamiento y optimización con GridSearch, alcanzando un Accuracy del 87%, Recall del 69% y Precision del 100%.
